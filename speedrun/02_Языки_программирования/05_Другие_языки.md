## Нет "лучшего" языка — есть подходящий

Языков программирования сотни. Каждый появился для решения конкретных проблем. Выбор языка зависит от задачи, команды, экосистемы.

---

## Python — универсальный швейцарский нож

**Где:** Data Science, Machine Learning, скрипты автоматизации, backend (Django/FastAPI).

**Почему популярен:**
- Простой синтаксис — легко читается как псевдокод
- Огромная библиотека для работы с данными (NumPy, Pandas, TensorFlow)
- Быстро писать, быстро прототипировать

**Минусы:** медленнее Java/Go, динамическая типизация даёт сюрпризы в больших проектах.

---

## JavaScript — язык веба

**Где:** Frontend (браузер) — единственный язык, который браузер понимает нативно. Backend (Node.js), мобайл (React Native).

**Почему важен:** без JS не было бы интерактивных сайтов. Любая кнопка, которая делает что-то без перезагрузки страницы — это JavaScript.

**TypeScript** — надстройка над JS со статической типизацией. Сейчас большинство серьёзных проектов на TS.

---

## Go (Golang) — для высоких нагрузок

**Где:** backend-сервисы с высокой нагрузкой, инфраструктура (Docker, Kubernetes написаны на Go).

**Почему:** компилируется в бинарный файл (быстро), встроенная поддержка конкурентности (горутины), простой язык без лишних сложностей.

## Итог

Не нужно знать все языки. Нужно понимать принципы — они переносятся между языками. Выучив один язык хорошо, второй учишь в разы быстрее.
