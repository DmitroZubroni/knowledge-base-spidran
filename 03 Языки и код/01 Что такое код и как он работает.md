## Процессор понимает только ноли и единицы

Внутри процессора — миллиарды транзисторов. Каждый в состоянии "включён" (1) или "выключён" (0). Это **машинный код** — единственное что процессор выполняет напрямую.

Писать в двоичном коде вручную невозможно. Поэтому придумали языки программирования.

---

## Язык программирования — это переводчик

Язык программирования позволяет писать инструкции в читаемом виде, которые затем превращаются в машинный код.

```
Ты пишешь:   System.out.println("Привет")
Компилятор:  превращает в байткод
JVM:         выполняет байткод на процессоре
```

---

## Основные конструкции — везде одинаковы

Синтаксис разный, идеи одни:

**Переменные** — именованные ячейки памяти:
```python
name = "Дмитрий"
age  = 25
```

**Условия** — ветвление логики:
```python
if age >= 18:
    print("Взрослый")
else:
    print("Несовершеннолетний")
```

**Циклы** — повторение действий:
```python
for i in range(5):
    print(i)   # 0, 1, 2, 3, 4
```

**Функции** — переиспользуемые блоки:
```python
def greet(name):
    return "Привет, " + name
```

## Типы данных

Каждое значение имеет тип — что это за данные:

| Тип | Примеры |
|-----|---------|
| `int` | 42, -7, 0 |
| `float` | 3.14, -0.5 |
| `String` | "текст", "имя" |
| `boolean` | true / false |
| `Array / List` | [1, 2, 3] |
| `Object / Map` | { key: value } |
