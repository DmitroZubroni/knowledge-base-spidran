## Приложение — это набор инструкций

Приложение (программа, software) — это файл с инструкциями для компьютера. Когда ты его запускаешь, процессор выполняет эти инструкции одну за другой.

**Аналогия:** рецепт блюда — это программа. Повар — процессор. Ингредиенты — данные. Готовое блюдо — результат.

---

## Из чего состоит приложение

Любое приложение — от калькулятора до соцсети — состоит из одних и тех же базовых компонентов:

```
┌─────────────────────────────────────┐
│              Приложение             │
├──────────────┬──────────────────────┤
│  Интерфейс  │     Логика           │
│  (что видит │  (правила и          │
│  пользователь) вычисления)         │
├──────────────┴──────────────────────┤
│              Данные                 │
│    (что приложение помнит)          │
└─────────────────────────────────────┘
```

- **Интерфейс** — кнопки, экраны, формы
- **Логика** — правила: "если нажали кнопку, то сделать X"
- **Данные** — то, что сохраняется между сессиями

---

## Программное обеспечение vs Железо

- **Hardware (железо)** — физические устройства: процессор, память, диск, экран
- **Software (ПО)** — программы, которые работают на железе
- **Операционная система** (Windows, macOS, Linux, Android) — программа которая управляет железом и запускает другие программы

Приложение не работает напрямую с железом — оно просит ОС это сделать.

---

## 💡 Ключевые понятия

| Термин | Значение |
|--------|---------|
| **Программа / приложение** | Набор инструкций для процессора |
| **Процесс** | Запущенная программа в памяти |
| **ОС** | Программа, управляющая железом |
| **Файл** | Данные, сохранённые на диске |
