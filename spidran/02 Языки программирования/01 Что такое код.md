## Компьютер понимает только ноли и единицы

Внутри процессора — миллиарды крошечных переключателей (транзисторов). Каждый может быть в состоянии "включён" (1) или "выключён" (0). Всё остальное — надстройка над этой физикой.

**Машинный код** — это команды прямо в виде битов: `10110000 01100001`. Никто не пишет так вручную.

---

## Языки программирования — переводчики

Язык программирования позволяет писать **человекочитаемые инструкции**, которые потом превращаются в машинный код.

```python
# Человек пишет:
print("Привет, мир!")

# Компьютер видит (грубо):
10110000 01100001 11000011...
```

---

## Алгоритм — это рецепт

**Алгоритм** — это последовательность шагов для решения задачи. Код — это алгоритм, записанный на языке программирования.

Пример алгоритма "приготовить яичницу":
```
1. Взять сковороду
2. Если нет масла → купить
3. Разогреть масло
4. Разбить яйцо
5. Ждать 3 минуты
6. Готово
```

В коде это: переменные, условия (if/else), циклы (повторить N раз), функции (переиспользуемые шаги).

---

## Переменная — это ящик с именем

```python
name = "Дмитрий"    # ящик с именем 'name', внутри строка
age = 25             # ящик 'age', внутри число
is_admin = True      # ящик 'is_admin', внутри да/нет
```

Функция — это именованный кусок кода который можно вызывать много раз:
```python
def greet(name):
    return "Привет, " + name + "!"

greet("Дмитрий")  # → "Привет, Дмитрий!"
greet("Мария")    # → "Привет, Мария!"
```
