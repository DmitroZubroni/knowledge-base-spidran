## CSS описывает как выглядят элементы

**CSS (Cascading Style Sheets)** — язык стилей. HTML отвечает за структуру, CSS — за внешний вид.

```css
/* Выбираем все теги <p> и задаём стили */
p {
  color: #333;
  font-size: 16px;
  line-height: 1.6;
}

/* Выбираем элемент с классом .button */
.button {
  background: #c0392b;
  border-radius: 8px;
  padding: 10px 20px;
}
```

---

## Каскад и специфичность

Название "каскадные" — потому что стили применяются по правилам приоритета:

```
!important > инлайн-стиль > id > class > тег
```

Это источник многих проблем у начинающих: стиль "не применяется" потому что перебивается более специфичным правилом.

## Блочная модель

Каждый HTML-элемент — прямоугольник:

```
┌───────────── margin ──────────────┐
│  ┌────────── border ───────────┐  │
│  │  ┌─────── padding ───────┐  │  │
│  │  │                       │  │  │
│  │  │      content          │  │  │
│  │  │                       │  │  │
│  │  └───────────────────────┘  │  │
│  └────────────────────────────┘  │
└───────────────────────────────────┘
```

- **content** — сам контент
- **padding** — внутренний отступ
- **border** — граница
- **margin** — внешний отступ

---

## Flexbox и Grid — современная вёрстка

Расположение элементов на странице — одна из главных задач CSS:

- **Flexbox** — для выравнивания в одном направлении (строка или колонка)
- **Grid** — для двумерной сетки (строки И колонки)

```css
.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}
```
