## IDE — умный редактор кода

**IDE (Integrated Development Environment)** — среда разработки. Не просто текстовый редактор — полноценный инструмент:

| Функция | Что делает |
|---------|-----------|
| **Автодополнение** | Предлагает варианты кода по контексту |
| **Рефакторинг** | Безопасно переименовать переменную во всём проекте |
| **Отладчик** | Остановить выполнение кода и посмотреть состояние |
| **Интеграция с Git** | Коммиты, diff, merge прямо в IDE |
| **Запуск тестов** | Запустить тесты, увидеть результат |
| **Линтер** | Подчеркнуть ошибки до запуска |

## Отладчик — главный инструмент при ошибках

**Debugger** позволяет:
- Поставить **breakpoint** — точку остановки
- При достижении — программа "замирает"
- Посмотреть значения всех переменных в этот момент
- Выполнять код пошагово (step by step)

Это на порядок эффективнее добавления `println` для отладки.

## Терминал — обязательный навык

Большинство инструментов разработчика работают через **командную строку (CLI)**:

```bash
# Запустить приложение
java -jar app.jar

# Установить зависимости
npm install

# Запустить в Docker
docker-compose up

# Посмотреть логи
tail -f app.log
```

IDE хороша для написания кода. Терминал — для управления проектом и сервером.
