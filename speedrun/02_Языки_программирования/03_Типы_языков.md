## Высокоуровневые и низкоуровневые

**Низкоуровневые** языки близки к железу. Ты управляешь памятью вручную, пишешь почти как процессор думает. Мощно и быстро, но сложно и опасно (легко сделать ошибку, которая уронит программу).

*Примеры: Assembler, C*

**Высокоуровневые** языки скрывают детали работы с железом. Ты думаешь о задаче, а не о памяти и байтах. Большинство современных языков — высокоуровневые.

*Примеры: Java, Python, JavaScript, Kotlin*

> Аналогия: низкоуровневый — ехать на велосипеде (контролируешь каждое движение), высокоуровневый — ехать на автомобиле с автопилотом.

## Статическая и динамическая типизация

**Тип данных** — это "вид" значения: число, текст, список, булево (да/нет).

**Статическая типизация** — типы указываются заранее и проверяются до запуска. Ошибки находятся на этапе написания кода.

```java
// Java — статическая типизация
int age = 25;        // это число
String name = "Дима"; // это строка
age = "привет";      // ❌ ОШИБКА — нельзя
```

**Динамическая типизация** — тип определяется во время выполнения. Гибче, но ошибки всплывают позже.

```python
# Python — динамическая типизация
age = 25
age = "привет"  # ✅ OK — переменная просто поменяла тип
```

**Строгая типизация** (Java, Python) — не даёт смешивать несовместимые типы.
**Слабая типизация** (JavaScript) — может неявно преобразовывать типы, иногда неожиданно.

## Парадигмы

**Парадигма** — это стиль программирования, подход к организации кода.

| Парадигма | Суть | Языки |
|-----------|------|-------|
| **ООП** (объектно-ориентированная) | Мир = объекты с данными и поведением | Java, C++ |
| **Функциональная** | Программа = набор функций без побочных эффектов | Haskell, Erlang |
| **Процедурная** | Программа = последовательность процедур | C, Pascal |

Большинство языков поддерживают несколько парадигм. Java — в первую очередь ООП, но поддерживает функциональный стиль.

## Итог

- Высокоуровневые языки = проще писать, ближе к человеку
- Статическая типизация = ошибки видно раньше
- Парадигма = стиль мышления при написании кода
